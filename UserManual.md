# 🎙️ Live MR Manager — 사용자 매뉴얼

**Live MR Manager**는 AI 기술로 어떤 노래든 보컬을 제거해 나만의 MR을 만들고, 라이브 환경에서 실시간으로 키·템포를 조절할 수 있는 도구입니다.

> 이 매뉴얼은 처음 사용하시는 분도 바로 따라 할 수 있도록 핵심 사용법 중심으로 정리했습니다.

---

## 🚀 1. 처음 시작하기 — AI 모델 설치

프로그램을 처음 실행하면 **AI 모델 파일**을 먼저 다운로드해야 합니다.

1. 왼쪽 메뉴에서 **[Settings]** 클릭
2. **[모델 다운로드]** 버튼 클릭 → 자동으로 최신 AI 분리 모델이 설치됩니다
3. 상태 배지가 **Online (Ready)** 로 바뀌면 준비 완료! ✅

> 💡 NVIDIA 그래픽카드(RTX 등)가 있으면 AI 분리 속도가 훨씬 빨라집니다.

---

## 🎵 2. 음악 추가하기

노래를 등록하는 방법은 두 가지입니다. 모든 노래는 추가 즉시 썸네일이 로컬에 자동 저장됩니다.

### 📺 유튜브에서 가져오기

1. 왼쪽 메뉴 **[YouTube]** 클릭
2. 유튜브 영상 주소(URL)를 입력창에 붙여넣기
3. **[정보 가져오기]** 클릭 → 정보 확인 후 자동으로 라이브러리에 추가됩니다

### 📁 내 파일 가져오기

1. 왼쪽 메뉴 **[My Files]** 클릭
2. `MP3`, `WAV`, `FLAC`, `M4A` 등 오디오 파일을 화면 중앙으로 **드래그 앤 드롭**

---

## 🤖 3. AI로 MR 만들기 (보컬 분리)

1. **[Library]** 에서 분리하고 싶은 곡을 찾습니다
2. 곡 위에서 **마우스 우클릭** → **[MR 분리]** 선택
3. 분리가 시작되며 왼쪽 **[Active Tasks]** 메뉴에서 진행률을 실시간 확인할 수 있습니다
4. 분리 중에도 다른 곡 재생, 라이브러리 탐색이 모두 가능합니다 (논블로킹 처리)
5. 완료되면 곡 카드에 **[MR]** 배지가 표시됩니다 ✅

> 취소하고 싶다면 **[Active Tasks]** 에서 해당 작업의 **[취소]** 버튼을 누르거나,  
> 곡 위에 우클릭 → **[분리 취소]** 를 눌러도 됩니다.

## 🎛️ 4. 재생 및 실시간 조절

화면 하단의 **컨트롤 바**에서 재생 관련 모든 것을 제어합니다.

| 컨트롤 | 설명 |
|---|---|
| **PITCH 슬라이더** | 노래 키를 -12 ~ +12 반음 단위로 조절 (수치를 클릭해 직접 입력 가능) |
| **TEMPO 슬라이더** | 재생 속도를 0.5x ~ 2.0x 범위로 조절 (수치를 클릭해 직접 입력 가능) |
| **VOLUME 슬라이더** | 전체 음량 조절 (수치를 클릭해 직접 입력 가능) |
| **VOCAL 스위치** | MR 분리 완료 곡에서 보컬 ON/OFF 실시간 전환 (300ms 선형 페이딩 적용) |
| **초기화 버튼** | 피치·템포·볼륨을 기본값으로 한 번에 초기화 |

> 💡 **고급 팁(v0.6.2)**: 
> - 우측의 **연두색 수치**를 클릭하면 키보드로 정확한 값을 직접 입력할 수 있습니다. 입력 시 레이아웃이 밀리지 않도록 고밀도 설계가 적용되었습니다.
> - 슬라이더를 **마우스 휠**로도 정밀 조절할 수 있습니다. **휠 클릭(중간 버튼)** 시 즉시 기본값으로 초기화됩니다.

## 📚 5. 라이브러리 관리

### 🔍 검색 및 필터

- **통합 검색창**: 제목, 아티스트, 태그로 동시에 검색
- **장르/카테고리 필터**: 드롭다운으로 원하는 장르나 카테고리만 보기
- **정렬**: 최근 추가순, 제목순, 재생 횟수순 등 선택 가능

### 📊 곡 정보 관리자 (Library Manager)

많은 곡을 한꺼번에 수정할 때 활용할 수 있는 **엑셀 스타일 일괄 편집 도구**입니다.

1. 상단 **[목록 관리]** 버튼 클릭
2. 표에서 제목, 아티스트, 카테고리, 장르, 태그를 직접 클릭해 수정
3. **[저장]** 버튼으로 한 번에 반영

### ✏️ 개별 곡 정보 수정

곡 위에서 **우클릭 → [정보 수정]** 을 누르면 상세 편집 창이 열립니다.

- 제목, 아티스트, 태그, 카테고리, 장르 수정
- **[온라인 검색]** 버튼으로 Last.fm 기반 자동 태그/장르 추가 가능
- MR 여부(플래그) 수동 설정 (AI 분리 완료 곡은 자동 잠김)

---

## 🏷️ 6. 태그 큐레이션 (번역 사전)

영어 태그를 한국어로 번역하거나, 나만의 태그 이름을 정의할 수 있는 기능입니다.

1. **[목록 관리]** → 상단 탭에서 **[Curation]** 선택
2. 아래 **미분류 태그 목록**에서 번역하고 싶은 항목을 클릭하면 원본 입력창에 자동으로 채워집니다
3. **번역어** 입력 후 **[사전에 등록]** 클릭
4. **즉시** 라이브러리 전체에 반영됩니다 — 새로 고침 불필요! 🎉

> 예: `soft` → `부드러운`, `ballad` → `발라드`
> 
> 💡 **태그 확장(v0.6.2)**: 팝 록, 어쿠스틱 록, K-팝 록 등 세분화된 장르와 자우림, 델리스파이스, 비스티 보이즈 등 주요 아티스트 태그가 시스템 사전에 대거 추가되어 실시간 번역을 지원합니다.

## 📁 7. 데이터 백업 및 복원

소중한 라이브러리 목록을 JSON 파일로 저장하고 복구할 수 있습니다.

| 기능 | 위치 | 설명 |
|---|---|---|
| **목록 백업** | Settings → [목록 백업] | 현재 라이브러리를 JSON 파일로 저장 |
| **목록 복원** | Settings → [목록 복원] | 백업 파일에서 없는 곡만 골라서 병합 |

---

## ⚡ 8. 방송 안전 모드 (Broadcast Safe)

AI 분리 작업 중 OBS 방송의 프레임 드랍을 방지하는 기능입니다.

- **[Active Tasks]** 또는 **[Settings]** 에서 **Broadcast Safe** 토글 ON
- 활성화 시 AI 연산 강도를 자동으로 조절해 방송 안정성을 우선시합니다

---

## 💡 자주 쓰는 단축키 및 팁

| 동작 | 방법 |
|---|---|
| 모달/메뉴 닫기 | `Esc` 키 |
| 슬라이더 기본값 초기화 | 슬라이더 위에서 **휠 클릭** |
| 곡 빠른 재생 | 카드 클릭 |
| 컨텍스트 메뉴 열기 | 카드 위에서 **우클릭** |
| 분리 취소 | Active Tasks → 취소 버튼 또는 카드 우클릭 → [분리 취소] |

## 📂 데이터 저장 위치

모든 데이터는 Windows의 로컬 앱 데이터 폴더에 안전하게 저장됩니다.

```
C:\Users\[사용자명]\AppData\Local\com.autumncolor77.live-mr-manager\
  ├── 라이브러리 DB (SQLite)
  ├── AI 모델 파일
  ├── 유튜브 썸네일 캐시
  └── 커스텀 태그 사전 (metadata_custom.json)
```
